# Durable Memory

This directory contains concise, human-editable memory shared by all agents.

- `project.md` — stable facts and conventions.
- `decisions/` — accepted decisions, one file per decision (`YYYY-MM-DD-type-slug.md`).
- `lessons/` — evidence-backed lessons, one file per project/incident (`YYYY-MM-DD-type-slug.md`).

File naming: `YYYY-MM-DD-type-slug.md`
- type: feat | fix | refactor | perf | docs | ci | chore

Do not store secrets, raw chain-of-thought, full transcripts, or temporary task chatter here.
